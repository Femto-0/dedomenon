File Upload Demo

🏗️ Architecture

A production-style full-stack application that extracts structured data using OCR + local Ollama LLM.

Upload → pdf-parse/OCR → Ollama (Llama3) → ExcelJS → Download

📦 Tech Stack

LayerTechnology
RuntimeNode.js (ES Modules)
ServerExpress + Multer
Processingpdf-parse + Tesseract.js
IntelligenceOllama (llama3)
OutputExcelJS

⚙️ Configuration

VariableDefaultDescription
PORT3000Server port
OLLAMA_URLhttp://localhost:11434Ollama API URL
OLLAMA_MODELllama3Model selection

🔌 API Reference

POST /api/upload

Request: multipart/form-data (Field: document)

Response: Structured JSON + Excel Download URL

🤖 Supported Types

TypeExtensionsMethod
PDF.pdfpdf-parse
Images.png, .jpg, .webpTesseract.js OCR
